# Product Overview

Camargue Sailing is a Flask-based web application for promoting and managing sailing voyage bookings in the Camargue area of South France.

## Core Features

- Public information pages about sailing voyages and the Camargue region
- User registration with email verification workflow
- Booking system for managing voyage reservations with calendar view
- Discussion forum for registered users to share experiences
- Email notifications for account verification and booking confirmations

## User Journey

1. Visitors browse public pages (home, voyages, Camargue info, voyage options)
2. Users register with email/password and verify via email link
3. Verified users can book sailing voyages and participate in forum discussions
4. System prevents overlapping bookings and manages availability calendar

## Business Context

The application supports a sailing business operating AMEL boats for one-week sailing trips in South France. The platform serves as both a marketing tool and operational booking system.
